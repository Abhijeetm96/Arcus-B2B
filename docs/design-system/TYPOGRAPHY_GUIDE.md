# ✍️ Typography Guidelines

Typography establishes content hierarchy, ensures reading comfort, and projects a professional tone.

---

## 1. Font Family Standard
* **Headings**: `Poppins` (Weights: 700/800 bold) - Projects strong industrial identity.
* **Body / Text**: `Inter` (Weights: 400/500/600 regular-semi) - Optimized for tabular data readability.

---

## 2. Text Sizing Scale

* **Display (H1)**: `64px` (`line-height: 110%`) - Brand hero headers.
* **Page Titles (H2)**: `24px` (`line-height: 120%`) - Workspace screen titles.
* **Section Titles (H3)**: `18px` (`line-height: 120%`) - Card headers.
* **Body Text**: `14px` (`line-height: 150%`) - Default lists, table text.
* **Caption / Label**: `12px` (`line-height: 100%`) - Input headers, meta stamps.
