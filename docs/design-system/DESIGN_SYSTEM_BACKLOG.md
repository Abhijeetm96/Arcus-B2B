# 📋 ARCUS Design System Backlog

This backlog lists deferred roadmap enhancements and technical debt items identified for future iterations of the ARCUS Design System. These are non-blocking items for the upcoming RFQ Workspace module.

---

## 1. Dark Mode Expansion
* **Description**: Fully propagate the `.dark` class toggles to all dashboard screens and customize contrast checks.
* **Priority**: Medium
* **Target Release**: v1.2

## 2. Custom Color Theme Switchers
* **Description**: Allow dynamic loading of tenant-specific brand colors by mapping CSS variables to theme custom configurations.
* **Priority**: Low
* **Target Release**: Future Phase

## 3. Drag-and-Drop File Upload Area
* **Description**: Extend the `<UploadComponent>` to support HTML5 drag-and-drop state indicators and drop files queue previews.
* **Priority**: Medium
* **Target Release**: v1.2

## 4. Extended Keyboard Accessibility Tests
* **Description**: Conduct automated screen reader tests (e.g. NVDA/VoiceOver) across complex multi-step forms.
* **Priority**: High
* **Target Release**: v1.1.1 (Verification)
