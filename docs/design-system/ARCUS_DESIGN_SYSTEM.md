# 🏗️ ARCUS Enterprise Design System (v1.0)

Welcome to the ARCUS Enterprise Design System. This system provides a unified visual language, reusable component architecture, and interaction standards to ensure the ARCUS platform delivers a professional, modern, and minimal industrial SaaS experience.

---

## 1. Visual Identity & Brand Values
ARCUS represents solidity, price transparency, and operational efficiency in construction procurement. The brand identity leans on:
* **Industrial Confidence**: Strong grays and solid layouts inspired by Caterpillar and JCB.
* **Premium Precision**: Vibrant golden-yellow highlights representing high quality.
* **Confidence & Speed**: Fast, clean interactions styled with minimal borders and subtle shadows.

---

## 2. Component Layout Hierarchy
All pages follow a single screen layout standard:
1. **Navbar** — Brand logo, global search, and cart summary.
2. **Sidebar** — Quick navigation by portals (Admin, Business, Individual, Professional).
3. **Breadcrumbs** — Contextual path trace (e.g. `Admin > RFQs > RFQ #1024`).
4. **Page Header** — Unified H1 title, detail description, and primary action CTAs.
5. **Workspace Grid** — Modular content cards, metrics widgets, and tables.
6. **Pagination & Footers** — Standard list navigation and support grids.
