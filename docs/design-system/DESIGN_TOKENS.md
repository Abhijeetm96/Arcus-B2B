# 🎨 ARCUS Design Tokens

Design Tokens are the visual atoms of the design system. Colors, typography, spacing, and shadows are defined as tokens to avoid arbitrary values.

---

## 1. Semantic Color Mappings

| Token Name | Value (Light) | Value (Dark) | Description |
| :--- | :--- | :--- | :--- |
| `primary` | `#fabd00` | `#fabd00` | Accent golden yellow. |
| `primary-hover` | `#e5ad00` | `#e5ad00` | Hover state for buttons. |
| `background` | `#f9f9f9` | `#121212` | Global page background. |
| `surface` | `#ffffff` | `#1e1e1e` | Containers and cards background. |
| `border` | `#e2e2e2` | `#333333` | Borders and lines. |
| `text-primary` | `#1c1b1b` | `#f9f9f9` | High contrast body text. |
| `text-secondary` | `#5f5e5e` | `#a5a4a4` | Subtitles and list details. |
| `success` | `#10b981` | `#10b981` | Completed states. |
| `warning` | `#f59e0b` | `#f59e0b` | Active negotiation states. |
| `danger` | `#ef4444` | `#ef4444` | Cancelled/Rejected states. |

---

## 2. Spacing & Sizing Scale

* **`4px`** (`space-xs`): Sub-element padding, labels margins.
* **`8px`** (`space-sm`): Standard gap inside components.
* **`12px`** (`space-md`): Form field margins, badge spacing.
* **`16px`** (`space-lg`): Component padding, list items gap.
* **`24px`** (`space-xl`): Page sections padding, grid gaps.
* **`32px`** (`space-xxl`): Larger margins, workspace gaps.
