# 🎖️ ARCUS BuildPoints Program

ARCUS features a loyalty program called **BuildPoints** to reward users for materials procurement and booking maintenance contractors.

## 📈 Point Accrual System

- **Retail Accrual Rate**: Individual customers earn **1 BuildPoint for every ₹100 spent** on building materials.
- **B2B Accrual Rate**: Corporate builders earn **1 BuildPoint for every ₹200 spent** on bulk materials.
- **Contractor Accrual Rate**: Earn points by completing bookings.

---

## 🎁 Point Redemption & Coupons

Points can be redeemed in the User Dashboard settings for custom promo codes:
- **WELCOME5**: Redeemable by individual customers (B2C) for a 5% discount on checkout orders.
- **ARCUS10**: Redeemable by verified business builders (B2B) for a 10% discount on B2B orders.
